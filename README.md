This program is a C++ implementation of the Map Reducer. 

To run on Linux / WSL:
- make clean
- make all
- ./bin/main shakespeare/ output/ temp/
    - structure of command line input:
        - ./bin/main <input_dir (default = "shakespeare")> <output_dir (default = "output")> <temp_dir (default = "temp")>
